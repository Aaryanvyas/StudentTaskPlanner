# Smart Student Task Planner

## 📌 Project Overview
The Smart Student Task Planner is a console-based Java application designed to help students manage their daily academic tasks efficiently. It allows users to organize tasks, assign priorities, track completion status, and store data permanently using file handling.

This project demonstrates the use of core Java concepts such as Object-Oriented Programming (OOP), file handling, and user interaction through a menu-driven interface.

---

## 🎯 Problem Statement
Students often struggle to manage multiple assignments, deadlines, and daily tasks. Traditional methods like writing tasks on paper are not reliable and can lead to missed deadlines.

This project provides a simple digital solution to manage tasks efficiently.

---

## 💡 Features

- ➕ Add new tasks with priority (High / Medium / Low)
- 📋 View all tasks in a structured format
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Save tasks using file handling (data persistence)
- 🔄 Load tasks automatically when program starts
- 🖥️ Simple console-based user interface

---

## 🛠️ Technologies Used

- **Java** (Core Java)
- **Object-Oriented Programming (OOP)**
- **File Handling (BufferedReader, FileWriter)**
- **VS Code**

---

## 🧠 Concepts Used

This project demonstrates:

- Classes and Objects  
- Encapsulation  
- ArrayList usage  
- File handling (read/write operations)  
- Looping and conditional statements  
- Menu-driven programming  

---

## ⚙️ How It Works

1. User runs the program  
2. A menu is displayed with options  
3. User selects an option:
   - Add task  
   - View tasks  
   - Mark task complete  
   - Delete task  
4. Tasks are stored in a file (`tasks.txt`)  
5. When program restarts, tasks are loaded automatically
